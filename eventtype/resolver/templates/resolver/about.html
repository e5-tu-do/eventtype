LHCb Event Type Explainer

Online tool to help ease the confusion that comes with LHCb event type.

The goal is to provide both

  • a tool that deciphers the meaning of a given event type
  • and a wizard that helps find the correct event type of a decay.
The software is provided without any promise of correctness. Please report any bugs you find on Github :)
